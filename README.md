# Multi-Agent Travel Planner
A Multi-Agent Travel Planner built using Python and Google ADK. The system uses multiple AI agents to work together and help users to plan their trips efficiently.

## Features
- Multi-agent architecture using Google ADK
- AI-powered travel planning
- Dedicated Planner Agent
- Modular and extensible agent design
- Built using Python

## Technologies Used
- Python
- Google ADK
- Gemini API
- Git
- GitHub

## How to run
1. Clone the repository.
2. Create a Python virtual environment.
3. Install the required dependencies.
4. Add your Google Gemini API key to '.env' file.
5. Run the project.
